# Frontend

Aplicação frontend para gerenciamento de clientes.

## Tecnologias usadas

- React
- Vite

## Requisitos

- Aplicação [backend](../backend/README.md) rodando na porta 8080
- Node.js 20

## Quick start

```bash
npm install
npm run dev
```

A aplicação estará disponível em `http://localhost:3000`.

## Rodando testes

```bash
npm run test
```
